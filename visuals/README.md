# Visual Insights Report

This folder contains the consolidated visual deliverable for the project.

## Overview

The PDF report presents the key analytical findings derived from the SQL analysis and summarized in the project README.

Visualizations are designed to:
- Highlight geographic and operational trade-offs
- Illustrate revenue concentration patterns
- Support customer behavior and RFM insights

## How to Read the Report

Each section of the report corresponds directly to one of the four key business insights presented in the main README.

The visuals are intended to complement the written analysis by providing intuitive, decision-oriented representations of the underlying data.

For detailed context and interpretation, refer to:
- `README.md`
- `/docs/executive_summary.md`
- `/docs/business_diagnostic.md`
